<h1>E-commerce Platform</h1>

<h1>Overview</h1>
<p>An e-commerce platform that allows users to purchase electronic products, and includes many functions <br>such as a real-time chat application, and a virtual payment gateway.<br>
  This platform is built by (React.js , Node.js , MongoDB and Express.js)</p>
<h1>Features</h1>
<ul>
   <li>
    <h2>Admistration functionalities</h2>
     <p>- The admin can </p>
    <p>- brows all users accounts  </p>
    <p>- brows all product and update (CRUD operations )  </p>
  </li>
  <li>
    <h2>E-commerce functionalities</h2>
    <p>- users can brows products , add to cart and buy </p>
  </li>
   <li>
    <h2>Braintree Payment Gateway</h2>
    <p>- the project wrap an payment gateway using Braintree gateway </p>
  </li>
   <li>
    <h2>Real time chat app</h2>
    <p>- user can send and recieve messages from other users in real time using Socket.io </p>
  </li>
</ul>

<h1>Tech Stack</h1>
<ul>
  <li>
    <h2>Frontend</h2>
    <p>React.js with Tailwindcss </p>
  </li>
  <li>
     <h2>Backend</h2>
     <p> Node.js and Express.js for backend functionalities</p>
  </li>
  <li>
    <h2>  Databases</h2>
    <p> MongoDB for storing products, users,purchases and  messages </p>
  </li>
</ul>
